# VisionTab

Multimodal tab complete for your editor.

## Overview
VisionTab is a multimodal tab completion model designed to provide intelligent, context-aware code suggestions. 

### Model Details
- **Base Model:** Gemma 4
- **Techniques:** Fine-tuned using **LoRA** (Low-Rank Adaptation) and **FIM** (Fill-In-the-Middle) to effectively learn code insertion and completion tasks.
- **Training Hardware:** 2x T4 GPUs
- **Dataset Size:** 10,000 samples
